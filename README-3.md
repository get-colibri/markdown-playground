# Colibri — Collaborative Markdown Editor for GitHub

Colibri is an online **collaborative Markdown editor** designed specifically to work with **GitHub repositories**. It lets teams edit `.md` files together in real time and sync changes back to GitHub as clean pull requests.

## 📌 Key Purpose

Colibri helps teams **edit Markdown files together** without leaving GitHub workflows.
It brings the ease of collaborative editing (like Google Docs) to Markdown files stored on GitHub.

## 💡 What You Can Do

### ✍️ Live Collaboration

Work together with others in real time on any Markdown file.

### 🔗 GitHub Integration

Open any `.md` file from a GitHub repo and edit it live — then merge changes back as a pull request.

### 👥 Share Easily

Share the editor with others instantly. No account required for participants — just send the link.

### 💬 Inline Comments

Add comments directly in the document for asynchronous reviews and discussion.

## 📊 Features

- **Realtime editing** with teammates.
- **Shareable editor links** for teammates or collaborators.
- **Git sync** — push updates back into GitHub via pull requests.
- **Inline comments** for review workflows.
